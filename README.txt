MU H7 - Versión offline (simple)
---------------------------------
Contenido:
- index.html : archivo principal del juego.
- assets/ : imágenes para héroes, monstruos y fondo.

Cómo jugar:
1. Descomprime la carpeta mu-h7-offline si está en ZIP.
2. Abre index.html en tu navegador (Chrome, Edge o Firefox).
3. Selecciona un héroe y juega offline. Los héroes atacan automáticamente cada 2s.
4. Usa 'Cambiar Skin' para alternar apariencias.

Opciones avanzadas:
- Para convertir en ejecutable: podemos empacarlo con Electron o crear un proyecto en Godot y exportar.
- Si quieres más héroes, skins o funcionalidades online, dime y lo amplío.
